### Keybinds

  `Space`: Toggle pause/play

  `Left/Right`: Seek backward/forward by 5 seconds

  `Ctrl + Left/Right`: Previous/next track

  `Up/Down`: Adjust volume

  `Mouse wheel`: Adjust widget opacity
